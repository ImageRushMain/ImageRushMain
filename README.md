# Welcome to ImageRush Organization! 🚀

We are the core team behind **ImageRush**, a high-performance, 100% in-browser graphic design engine built with Next.js and PixiJS. 

Our mission is to empower creators to generate stunning thumbnails, banners, and overlays instantly. By leveraging real-time WebGL shaders, cinematic animations, and a rich template marketplace, we're pushing the boundaries of what's possible directly in the browser—with zero server-side rendering bottlenecks.

Follow our repositories as we build the future of high-fidelity web rendering! ✨
